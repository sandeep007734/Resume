 \normalsize{\textsc{Modeling Storage Performance in a HPC System using Machine Learning.}} \textbf{[ME Thesis]}\\
\footnotesize{We present a mathematical model that can capture the relationship between the \textit{features} (configuration parameters of a file system, hardware configuration and the workload configuration) and the \textit{performance metrics} (Read speed, write speed of disk etc.) and use this to rank the features according to their importance in deciding the performance of the parallel file system (Gluster FS).}\\\phantom{.}\hfill\small Advisor: \href{http://drona.csa.iisc.ernet.in/~gopi/}{Prof. K. \textsc{Gopinath}} | \normalsize
